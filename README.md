### Для запуска проекта вам необходимо :

1. Клонировать проект на ваше устройство командой:
```javascript
git clone https://github.com/KasumovW/test-task-4.git
```

2. Указать путь к проекту:
```javascript
cd test-task-4
```

3. Затем устоновить зависимости проекта:
```javascript
npm install
```

4. Запустить:
```javascript
npm run dev
```
